# n° 2: Concessionario auto
Si consideri l'entità veicolo con i seguenti attributi:
- id (identificatore univoco del veicolo)
- modello (es. panda)
- marca (es. fiat)
- codice immatricolazione
- data immatricolazione
- prezzo
In base alla proprietà OOP ereditarietà, e nell’approccio MVC, scrivere tre classi model (automobile, motocicletta, e furgone) che sono figlie della classe padre veicolo.
1) Scrivere un’applicazione in java che implementa le classi sopra descritte e, data una lista di veicoli, permette di calcolare il prezzo medio, il veicolo disonibile meno caro e il prezzo medio tra i veicoli di una certa marca. Scegliere l’interfaccia (console,web,altro) in cui l’utente richiede le operazioni da fare sulla lista dei veicoli presenti nel Concessionario.
2) Scrivere un progetto utilizzando il framework Angular in grado di rappresentare una pagina web con tutte le schede dei veicoli appartenenti ad una certa categoria (es. motocicletta). Una pagina iniziale deve avere la lista delle categorie e, selezionata la categoria, deve apparire una pagina con tutti i veicoli di detta categoria. Utilizzare le components disponibili nella libreria bootstrap.
3) Scrivere la query in SQL che permette di selezionare tutti i veicoli aventi un prezzo inferiore ad un certo importo.
4) Scrivere un documento finale dove si descrivono i progetti sviluppati, gli ambienti di sviluppo usati per svilupparli ed i test che sono stati fatti alla fine della fase dello sviluppo.
Condizioni di sviluppo
Utilizzare il framework: Angular per il client e le components messe a disposizione da Bootstrap. Utilizzare il JDK installato sulla postazione assegnata per gli sviluppi in java. Utilizzare il DBMS MySQL per la parte di SQL. Scegliere l’IDE più opportuno per ognuno dei software richiesti.
